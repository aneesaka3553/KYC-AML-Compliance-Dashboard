# KYC-AML-Compliance-Dashboard
KYC &amp; AML Risk Analysis using Excel and Power BI

## 📌 Overview

This project presents an end-to-end KYC (Know Your Customer) and AML (Anti-Money Laundering) compliance dashboard designed to simulate real-world banking operations. The dashboard focuses on improving client onboarding efficiency, monitoring compliance status, and identifying high-risk customers through data-driven insights.

## 📊 Dataset

* 550+ client records
* 1000+ transaction entries
* Multiple attributes including geography, PEP status, income, and KYC documentation

## ⚙️ Tools & Technologies

* Microsoft Excel – Data preparation, validation, and risk modeling
* Power BI – Data visualization and interactive dashboard development

## 🧠 Key Features

### 🔹 KYC Monitoring

* Tracks KYC completion status (Completed, Pending, In Progress)
* Identifies documentation gaps and onboarding delays

### 🔹 Risk Segmentation

* Classifies clients into Low, Medium, and High-risk categories
* Uses a layered risk model incorporating:

  * Customer risk (PEP, income)
  * Geographic risk
  * Transaction risk
  * Behavioral indicators

### 🔹 Transaction Monitoring

* Flags suspicious transactions based on amount and risk geography
* Tracks transaction trends and anomalies over time

### 🔹 Interactive Dashboard

* KPI cards for key metrics (Total Clients, KYC %, High Risk, Flagged Transactions)
* Dynamic filters (Country, Risk Category, KYC Status)
* Visual insights using charts and tables

## 📈 Business Impact

* Enhances visibility into KYC compliance and onboarding bottlenecks
* Supports risk-based decision-making
* Improves identification of high-risk clients and suspicious activity
* Demonstrates how data analytics can strengthen regulatory compliance

## 🚀 Conclusion

This project showcases the application of data analytics in financial compliance, combining structured data, risk modeling, and visualization to simulate real-world AML monitoring systems used in banking institutions.
